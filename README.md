# 💳 Task‑3: ATM Interface

This project is part of my **Java Programming Internship at CodSoft**.  
It simulates an ATM system where users can interact with their bank account to perform basic operations.

---

## 📌 Features
- Check account balance.
- Deposit funds into the account.
- Withdraw funds with validation (ensures sufficient balance).
- Display appropriate messages for each transaction.
- Simple, user‑friendly console interface.

---

## 🛠️ Concepts Used
- Object‑Oriented Programming (OOP) principles.
- Class design (`BankAccount` and `ATMInterface`).
- Methods for deposit, withdraw, and balance check.
- Input validation using conditional statements.
- User input handling (`Scanner`).

---

## 🚀 How to Run
1. Save the file as `ATMInterface.java`.
2. Compile the program:
   ```bash
   javac ATMInterface.java
